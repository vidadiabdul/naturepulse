# Nature Pulse
A platform for nature enthusiasts to explore and book activities.

## Features
- Login/Register
- Multilanguage
- Add to favorite
- Search activities
- Filter activities by category and location.
- Responsive design.
- User-friendly UI.
- Contact form

## Technologies
- React.js
- Tailwind CSS
- PHP backend
